I created a dynamic single-page website for a fictional café that updates daily to provide a fresh and engaging experience. Using HTML, CSS, and JavaScript, the page rotates unique colors, images, and content for each day of the week, showcasing a new café item every day.

Features I implemented include:

Daily Themes: Each weekday highlights a specific café item (e.g., Bubble Tea for Monday), complete with a unique description mentioning the day.

Custom Imagery: Every day displays a distinct image of the highlighted item, with well-crafted alt tags for accessibility.

Dynamic Design: The background color changes daily to complement the featured item and enhance the visual experience.

Responsive Content: The page is designed to provide seamless updates with all elements integrated into one cohesive design.

This project demonstrates my ability to create interactive and visually appealing web pages, showcasing attention to detail, accessibility, and user engagement. Visit the page here to explore it for yourself!
